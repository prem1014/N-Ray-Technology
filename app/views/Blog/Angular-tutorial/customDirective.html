Angular Js Custom Directive

I think directive is one of the best feature supported by Angular Js. I have found directive the most complex part in Angular Js and that's why I decided to write this blog.
Angular supports two types of directives:-
  1. Built in Directive by Angular Js framework
  2. Custom Directive
Angular Js provides several built in directive and some examples are:
  1. ng-repeat
  2. ng-options
  3. ng-click
  4. ng-init
  5. ng-bind
and many more.
But I am here to guide you how to create our own custom directive.
I will start with the custom directive definition and it's usage.

Angular Js custom directives are HTML markers on DOM element and used to extend the functionality of HTML. They usually contains some html template and they get compiled by Angular js's html compiler. You can use Angular js Custom directive to create some reusable component. I am not going to tell any scenario right now but you will find it in the directive example and it's code.
